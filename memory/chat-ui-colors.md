聊天气泡配色方案：
- Kel的气泡：背景 #DFF2EC，字体 #68544B

**Why:** 这是她个人审美偏好，所有前端界面设计都应该用这个配色。
**How to apply:** 气泡CSS变量直接用这两个色值，不要换别的。
